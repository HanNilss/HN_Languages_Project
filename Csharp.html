Year of creation:

2000

Creator:

Microsoft

History:

C# was initially launched by Microsoft as a part of the .NET framework and was designed for Microsoft applications as a competitor to Java. It has througout the years been filled with more and more features and is now in version 11.

Use

Since C# is a general purpose programming language, it can be used for virtually anything. Some applications and programs that C# are commonly used for are video games, especially since the Unity game engine is based on it, mobile apps, and website development, helped by the .NET framework.

Current state:

Today C# is one of the most popular programming languages in the world, ranked amongst Java and JavaScript. Its popularity is heavily reliant on Microsoft, its creator, since most Microsoft applications is built with it. As it is the base language for the Unity game engine, C# is readily available for both experts and upstart game developers, which makes it very prominent in the game development scene.
